# Hippius CLI & SDK

:::warning Deprecated
The Hippius CLI and SDK are currently being rebuilt for the new Arion storage backend. The previous IPFS-based CLI is no longer functional.
:::

## For File Storage

Use any S3-compatible client to store and retrieve files on Hippius. See the [Quickstart guide](/use/quickstart) for setup instructions.

## For Miner Operations

If you're registering or managing miner nodes, see the [Miner Setup guide](/earn/arion/running-blockchain-node).

## Tracking the Rewrite

Follow progress on the CLI rewrite in the [GitHub repository](https://github.com/thenervelab/hippius).
